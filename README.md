## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

```
.env.local - values
NEXT_PUBLIC_API_URL = your value
NEXT_PUBLIC_DYNAMIC_URL= your value
```
